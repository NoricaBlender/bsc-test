Help / How to run

This runs entirely in your browser. No build step.

Option A — Run locally (recommended)

  1. Download the project folder and unzip it.
  2. Serve it with any static server (because browsers block ES module imports from file://).
python -m http.server 8000

Then open: http://localhost:8000

Option B — Host on GitHub Pages

  1. Create a new GitHub repo and upload the folder contents.
  2. Repo → Settings → Pages → Deploy from branch → choose main / root.
  3. Wait for GitHub to publish the site, then open the Pages URL.

Controls

  • Builder: click/tap to place. Hold Shift + click/tap to remove.
  • Battle (desktop): WASD to move, mouse to aim, Space to fire.
  • Battle (mobile): joystick to move, Fire button to shoot.

Notes

  • Your ship save is stored in localStorage in your browser.
  • The buoyancy model is simplified: each block samples the water plane and applies lift/drag.
